How to Use
----------
1. SassTray-*-bundle-ruby-*-sass-*.zip をダウンロードする
2. Zipファイルを展開する
3. SassTray フォルダにある SassTray.exe に監視したいフォルダをドロップする

Config
------
監視するフォルダの中にSassTray.configというファイルを置くことで各種設定(出力先やSassコマンドのオプション)を変更できます。
